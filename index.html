    How Does
    Xerom work with ethoFS?

    The Xerom project uses the same ethoFS file delivery technology as its Ether-1 parent project. Just like Ether-1, Xerom has its own completely independent chain. For ours users, a new level of redundancy is achieved by using both chains to link to ethoFS content. The Xerom blockchain and ethoFS implementation allow the developers of both projects to rapidly test new feature and roll up all mature technology improvements into the Ether-1 project.
